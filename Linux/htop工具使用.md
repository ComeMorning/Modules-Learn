#### CentOS7 安装

```sh
yum install epel-release -y
yum install htop -y
```

#### Centos6 安装

```sh
rpm -ivh https://mirrors.tuna.tsinghua.edu.cn/epel/6/x86_64/epel-release-6-8.noarch.rpm
rpm --import /etc/pki/rpm-gpg/RPM-GPG-KEY-EPEL
yum install htop -y
```
